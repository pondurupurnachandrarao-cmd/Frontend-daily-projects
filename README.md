# 👤 Personal Profile Card

A simple and responsive personal profile card built using **HTML** and **CSS**. It showcases a profile image, name, role, short introduction, social media links, and action buttons.

## 🚀 Features
- Responsive Profile Card
- Clean & Modern UI
- Social Media Links
- Contact & Resume Buttons
- Beginner-Friendly Project

## 🛠️ Technologies Used
- HTML5
- CSS3
