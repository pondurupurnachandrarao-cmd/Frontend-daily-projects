# 🌐 Personal Portfolio Website

A modern, responsive, and professional **Personal Portfolio Website** built using **HTML5** and **CSS3**. This project is designed to showcase a developer's profile, technical skills, projects, services, resume, and contact information in a clean and visually appealing layout.

The portfolio serves as a personal branding website where recruiters, clients, and employers can quickly learn about your background, explore your work, and get in touch with you.
